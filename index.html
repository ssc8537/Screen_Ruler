测量显示器的真实尺寸

用 A4 纸、身份证/银行卡、人民币纸币或硬币等常见实物校准后，得到显示区域的长、宽与对角线，单位为现实生活中的厘米。

01

一次性校准

校准实际长度

把 A4 纸短边对折，使用折叠后的一半宽度。纸张越平整、对齐越仔细，结果越准。

  1. 选择你手边最容易找到、边缘最直的标准尺寸实物。
  2. 将实物对应的边或直径贴在屏幕上的蓝色标尺上。
  3. 拖动滑块，直到标尺两端与实物长度完全对齐，再保存校准。

校准后这里会出现一把真实厘米直尺，可以调整长度，用来测量身边的小物品。

把物品贴进屏幕下沿或直尺边缘，从 0 开始读数。若浏览器缩放或系统缩放变化，请重新校准。

为什么需要校准？

网页中的厘米不一定是真实厘米

浏览器会因系统缩放、显示器像素密度和缩放比例而改变 CSS 像素的物理大小。此工具用你亲自对齐的实物建立"每厘米对应多少屏幕单位"的比例，因此计算结果对应真实长度。
