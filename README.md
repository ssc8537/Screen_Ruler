# 真实屏幕尺子

一个无需安装、无外部依赖的 HTML 工具，用于测量当前显示器**显示区域**的真实物理宽度、高度和对角线，主单位为厘米。
完成校准后，它也可以显示一把真实厘米直尺，用电脑屏幕直接测量身边的小物件。

## 使用

1. 在浏览器中以 **100% 缩放**打开 `index.html`。
2. 选择一种手边有的标准尺寸实物。推荐优先使用 **A4 纸宽度的一半（10.50 cm）**，也可以用 A4 全宽、人民币纸币、身份证/银行卡或硬币。
3. 把实物对应的边或直径贴在蓝色校准标尺上。
4. 拖动滑块，使蓝色标尺的两端正好与实物长度重合。
5. 点击“保存校准并测量”。
6. 在“屏幕直尺”区域调整尺子长度，把物品从 0 刻度开始贴着直尺边缘读取。

也可以选择“自定义已知长度”，使用任意有确定长度的实物进行校准。

校准数据只会保存到当前浏览器的 `localStorage`，不会上传。更换显示器、改变系统显示缩放比例或浏览器缩放比例后，需要重新校准。

## 实现原理

浏览器的 CSS `cm` 并不保证是现实中的 1 cm。此工具让用户把一个已知物理长度映射到屏幕的浏览器坐标：

```text
每厘米屏幕单位 = 校准标尺的像素长度 / 实物长度（cm）
屏幕宽（cm） = screen.width / 每厘米屏幕单位
屏幕高（cm） = screen.height / 每厘米屏幕单位
对角线（cm） = √(宽² + 高²)
```

校准标尺和 `screen.width` / `screen.height` 使用同一浏览器坐标体系，因此即使系统启用了显示缩放，实物校准仍能补偿该差异。

屏幕直尺同样使用保存后的校准比例：

```text
直尺显示宽度（px） = 直尺长度（cm） × 每厘米屏幕单位
```

直尺会按 1 mm 生成短刻度、0.5 cm 生成中刻度、1 cm 生成长刻度和数字。

## 内置校准物

优先选择更长、更平整、边缘更直的物品。硬币可用，但因为直径很短，手动误差会被明显放大。

| 参照物 | 长度 |
| --- | ---: |
| A4 纸宽度的一半 | 10.50 cm |
| A4 纸短边/宽度 | 21.00 cm |
| A4 纸长边 | 29.70 cm |
| 身份证/银行卡长边 | 8.56 cm |
| 身份证/银行卡短边 | 5.40 cm |
| 100 元人民币长边 | 15.50 cm |
| 50 元人民币长边 | 15.00 cm |
| 20 元人民币长边 | 14.50 cm |
| 10 元人民币长边 | 14.00 cm |
| 5 元人民币长边 | 13.50 cm |
| 1 元人民币纸币长边 | 13.00 cm |
| 2019 版 1 元硬币直径 | 2.225 cm |
| 旧版 1 元硬币直径 | 2.50 cm |
| 5 角硬币直径 | 2.05 cm |
| 1 角硬币直径 | 1.90 cm |

尺寸来源参考：[A 系列纸张尺寸](https://en.wikipedia.org/wiki/Paper_size#A_series)、[ISO/IEC 7810 ID-1 卡片尺寸](https://en.wikipedia.org/wiki/ISO/IEC_7810)、[人民币纸币与硬币公开资料](https://en.wikipedia.org/wiki/Renminbi)。实际使用时，如果实物磨损、卷曲、打印裁切不准，会影响校准结果。

## 调研摘要

GitHub 的同类项目以“像素标尺”为主；本项目吸收其交互思路，但把重点放在真实物理长度的校准上。

| 项目 | 借鉴的思路 |
| --- | --- |
| [Bluegrams/ScreenRuler](https://github.com/Bluegrams/ScreenRuler) | 提供二维尺、厘米/英寸、对角线和标记线；本工具采用其“二维尺寸 + 对角线”的结果表达。 |
| [EdiWang/Ruler](https://github.com/EdiWang/Ruler) | 重视高 DPI 屏幕支持；本工具通过实物校准避免依赖固定 DPI 假设。 |
| [MichaelAquilina/Windows-Pixel-Ruler](https://github.com/MichaelAquilina/Windows-Pixel-Ruler) | 明确区分屏幕像素测量与物理长度测量；本工具补足前者没有的厘米校准。 |
| [epilys/rlr](https://github.com/epilys/rlr) | 可交互标尺、精细移动和量角模式；本工具保留清晰、可调整的量尺交互。 |
| [Genymobile/screen-ruler](https://github.com/Genymobile/screen-ruler) | 将屏幕测量结果即时显示在轻量界面；本工具以实时结果卡片呈现长、宽和对角线。 |

“所有相似项目”无法穷尽，因为 GitHub 搜索结果会持续新增且包含大量名称相近、但实际用途不同的仓库。本调研覆盖了以 `screen ruler`、`screen measurement`、`屏幕尺子`、`屏幕测量` 为关键词得到的代表性、功能相关项目。
