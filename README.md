# 屏幕测量工具 - 屏幕尺子 (Screen Ruler)

一个简单易用的屏幕测量工具，基于 HTML5 Canvas，就像真实的尺子一样！

## 功能特性

- **直尺模式**：可自由拖拽、旋转、延伸的线段尺，带厘米刻度
- **矩形模式**：可调整大小的矩形框，自动显示宽、高、对角线长度
- **校准系统**：支持多种校准方式，确保测量准确
- **一键测量**：快速测量整个屏幕或对角线
- **起点/终点可拖出屏幕**：方便测量对角线

## 快速使用

1. 直接打开 `index.html` 文件
2. 双击用浏览器打开（**推荐使用 Edge 浏览器**）
3. **按 `F11` 键全屏浏览器**，这样就可以测量整个屏幕了！
4. 开始测量！

## 使用说明

### 重要提示
- **推荐使用 Edge 浏览器**
- 按 `F11` 键全屏浏览器，测量整个屏幕更方便

### 直尺模式
- 拖拽红色起点 - 移动直尺位置
- 拖拽蓝色终点 - 改变长度和角度
- 拖拽尺身 - 整体移动直尺
- 按住 Shift - 角度吸附（0°/45°/90°等）

### 矩形模式
- 拖拽四角手柄 - 调整矩形大小
- 拖拽边框 - 移动矩形位置
- 自动显示宽、高、对角线长度

### 快捷键
- `R` - 切换到直尺模式
- `M` - 切换到矩形框模式
- `C` - 打开校准面板
- `0` (数字零) - 重置当前工具
- `Shift` - 直尺角度吸附
- `Esc` - 关闭帮助面板、取消拖拽

## 校准方法

为了确保测量准确，建议首先进行校准：

1. 点击 `校准` 按钮
2. 选择校准方式：
   - 银行卡校准（推荐）
   - 身份证校准
   - A4纸校准
   - 硬币校准
   - 手动测量校准
   - 直接输入参数
